Configuration Help

Every Veriplace application, including this example, must specify its OAuth credentials and a few other parameters in order to access the Veriplace APIs.
By default, this example reads these parameters from a property file located in: 
/tmp/veriplace-example.properties
Alternate locations may be specified by modifying the example.

Expected Properties

veriplace.application.consumer.key
The application's OAuth consumer_key
veriplace.application.consumer.secret
The application's OAuth consumer_secret
veriplace.server URI
The url of Veriplace, typically http://veriplace.com
veriplace.application.token.value (optional)
An application-specific access token's value, required when calling the User Discovery API with a mobile number or email address.
veriplace.application.token.secret (optional)
An application-specific access token's secret, required when calling the User Discovery API with a mobile number or email address.
googleApiKey (optional)
An API key for google maps. Get one here.
