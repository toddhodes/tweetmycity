Tag require-location-permission

Requests permission for Veriplace to locate the current user, but doesn't actually acquire a location. The rest of the page will not execute until permission has been obtained; will redirect to an error page if the request fails. Use this tag if you want to identify the user and obtain permission ahead of time, when the actual location request will be made by a different page. The tag returns the current Veriplace user object.
